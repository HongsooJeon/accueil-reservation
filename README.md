# カフェ アクイーユ 恵比寿　予約管理表　自動作成ツール

ebica からダウンロードした PDF を置くだけで、  
当日の予約管理 Excel を自動で作ってくれるツールです。

---

## 毎日の使い方（3ステップ）

### STEP 1　ebica から PDF をダウンロードする

1. ebica の管理画面にログインする
2. 当日の予約一覧ページを開く
3. ページ右上などにある **「印刷」または「PDF出力」ボタン** を押す
4. ダウンロードされた PDF ファイルを確認する

---

### STEP 2　PDF を所定のフォルダに置く

ダウンロードした PDF ファイルを、以下のフォルダに移動（またはコピー）する。

```
accueil-reservation
└── input　　←　ここに PDF を入れる
```

> **ポイント**
> - フォルダの中に PDF が複数あっても大丈夫です。最後に追加したファイルを自動で読み込みます。
> - ファイル名は変えなくて構いません。

---

### STEP 3　ツールを起動する

デスクトップの **「予約管理　起動」** をダブルクリックする。

黒い画面（コマンドプロンプト）が開き、自動で処理が始まります。  
数秒後に以下のメッセージが出たら完了です。

```
*** 完了 ***
   出力ファイル: C:\...\output\アクイーユ_予約管理_20260417.xlsx
```

---

### STEP 4　Excel ファイルを開く

`output` フォルダの中に Excel ファイルが作成されています。

```
accueil-reservation
└── output
    └── アクイーユ_予約管理_20260417.xlsx　　←　これを開く
```

---

## Excel の見方

Excel には **3枚のシート** があります。

---

### シート 1「予約管理」

当日オペレーション用の一覧表です。

| 列 | 内容 |
|----|------|
| A：席 | テーブル番号 |
| B：時間 | 来店時間 |
| C：名前 | お客様のお名前（カナ） |
| D：人数 | 来客人数 |
| E：媒体 | 予約経路（OZmall・食べログ・直接　など） |
| F：割引・ポイント | 会計時に適用する割引・ポイントの金額 |
| G：プラン | ご予約のプラン名 |
| J：アレルギー | アレルギー・苦手食材の内容 |
| K：記念日メッセージ | プレートへのメッセージ内容 |
| L：備考・要望 | その他ご要望 |
| M：目的 | 誕生日・記念日・女子会　など |

> **行の色について**
> - 薄い赤（ピンク系）の行 → **会計時に割引・ポイント適用あり**　要注意
> - 薄い黄色の行 → **アレルギーあり**　対応が必要
> - 青系の行 → ランチ
> - 紫系の行 → ディナー

---

### シート 2「会計注意リスト」

**割引・ポイントが発生するお客様だけ**を抜き出した一覧です。  
会計担当スタッフがレジ対応前に確認するためのシートです。

> ヒトサラからのご予約は「当日ポイント利用の可能性あり」として掲載されます。  
> 来店時にお客様へポイント利用の有無をご確認ください。

---

### シート 3「AT集計（製造用）」

**いちごみるくアフタヌーンティーセット**の製造数一覧です。  
キッチンスタッフが仕込み数を確認するためのシートです。

時間帯ごとに「何台（何名分）作るか」が一目でわかります。  
アニバーサリーランチは別枠で集計されています。

---

## よくあるトラブル

### 「PDFが見つかりません」と表示された

→ `input` フォルダに PDF が入っているか確認してください。

---

### 「ファイルを保存できませんでした」と表示された

→ 前回作成した Excel ファイルを Excel で開いたままになっている可能性があります。  
　 Excel を閉じてから、もう一度ツールを起動してください。

---

### 予約データが 0 件と表示された

→ PDF が正しい ebica の出力かどうか確認してください。  
　 別のページや別のファイルをダウンロードしてしまっている場合があります。

---

### 特定の日付のファイルを作り直したい

`input` フォルダに該当日の PDF を入れて、黒い画面で以下のように入力してください。

```
python main.py --date 20260417
```

（`20260417` の部分を対象の日付に変えてください）

---

## フォルダ構成（参考）

```
accueil-reservation
├── input          ← ebica の PDF をここに置く
├── output         ← 完成した Excel はここに保存される
├── main.py        ← ツールの本体（触らない）
└── README.md      ← このファイル
```
